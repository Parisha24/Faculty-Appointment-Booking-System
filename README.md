# Facultyâ€“Student Appointment Management System

A web-based system that allows students to book appointments with faculty members and enables faculty to manage availability and approve or reject appointment requests.

---

## Features

### Student
- Student registration and login
- Student dashboard with name and enrollment ID
- Book new appointments
- View appointment history
- Track appointment status (Pending / Approved / Rejected)
- View faculty remarks for rejected appointments

### Faculty
- Faculty registration and login
- Faculty dashboard
- Create appointment time slots
- View student appointment requests in table format
- Approve or reject appointments
- View approved and rejected appointment history

---

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP
- **Database:** MySQL
- **Server:** Apache (XAMPP / WAMP)
